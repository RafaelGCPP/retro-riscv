# Feature Plans

This directory stores architecture-first plans for PO-prioritized features.

## Workflow
1. Create a folder: `docs/feature-plans/<feature-slug>/`
2. Fill required plan files (`00` to `04`).
3. Hand off implementation using prompts in `03-dev-agent-prompts.md`.
4. Ensure post-implementation docs are updated.
5. Validate no contract/documentation drift.

## Naming
Use lowercase kebab-case for `<feature-slug>`.
Example: `uart-rx-interrupts`
